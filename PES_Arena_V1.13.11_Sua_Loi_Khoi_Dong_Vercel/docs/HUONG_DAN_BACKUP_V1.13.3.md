# Hướng dẫn Backup dữ liệu V1.13.3

1. Đăng nhập tài khoản có quyền **Sao lưu dữ liệu**.
2. Vào **Admin → Backup dữ liệu**.
3. Nếu đang chạy Production, nhập chính xác: `SAO LUU DU LIEU`.
4. Bấm **Tạo và tải Backup ZIP**.
5. Chờ file ZIP tải xuống và lưu ở ổ đĩa riêng.

`SAO LUU DU LIEU` là câu xác nhận để tránh bấm nhầm, không phải mật khẩu. Backup chỉ đọc dữ liệu và không xóa hoặc thay đổi dữ liệu Supabase.

File Backup có thể chứa thông tin tài khoản, lịch sử trận và nhật ký. Không tải lên GitHub, không gửi công khai.
